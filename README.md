# Huarun-more-h5
Vite2.0 搭建的适配移动端项目，使用 `flexible` 方案。

+ 启动项目：

  ```js
  npm run dev
  ```

- 主要技术栈：`Vue3.0`   `Vite2.0`   `Vant3.0`   `flexible`
